Pseudoclases y Pseudoelementos

En este ejercicio estoy mostrando diferentes selectores de CSS y como puedo usarlos, en la descripción puedo mostrarte como estan definidos en esta misma página

Selectores de tipo

Son los que seleccionan una etiqueta según su tipo (Div, H1, P, Section, etc). En este ejemplo seleccioné el body para cambiarle el color y el padding a 50px asi como también cambiar la tipografia y tamaño a los textos.

Documentación

Selectores de clase

Son los que se seleccionan usando el atributo HTML "Class" y colocandole un nombre a ese class y luego relacionandolo en la hoja de estilos como .nombreClase, en este ejemplo estoy seleccionando las secciones de cada explicación de selector con el nombre de clase "seccion-explicacion" y agregandole a este selector atributos de border, border-radius, padding y margin.

Documentación

Selectores de ID

Son los que se seleccionan usando el atributo HTML "ID" y colocandole un nombre unico a ese id y luego relacionandolo en la hoja de estilos como #nombreID, en este ejemplo estoy seleccionando el titulo principal de la página y agregandole un atributo de text-decoration como underline.

Documentación

Selectores de Atributo

Son los que se seleccionan usando el valor de algún atributo HTML determinado. En la hoja de estilo se especifcia la etiqueta que se quiere seleccionar y luego entre corchetes [] el par atributo valor que se quiere identificar para seleccionar. En este ejemplo todos las etiquetas anchor tienen el mismo estilo por el selector de tipo, pero solo el hipervinculo del final tiene un estilo distinto gracias a este tipo de seleccion (estoy agregando un hipervinculo al primer elemento de la pagina que es el titulo principal identificado con el ID="titulo-principal").

Documentación

Selector universal

Es el que selecciona a todos los elementos HTML indistintivamente de su tipo y le agrega valores iniciales que se aplican en todo momento en la página, usualmente se usa como práctica de formatear todo los valores iniciales y que no generen conflicto con los distintos navegadores de internet.
