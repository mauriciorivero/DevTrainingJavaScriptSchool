Display Flex

En este ejercicio estoy mostrando diferentes modos de posicionamiento de elementos usando la propiedad CSS display flex

Display Flex - Default

En el display flex (solamente declarado) todos los elementos se ubican de manera lineal horizontalmente pegados unos a otros de izquierda a derecha

NOTA IMPORTANTE

Hay que tener en cuenta que flex-direction lo que hace es invertir los ejes o girarlos de row a column, por lo que las propidedades justify-content y align-items se deben usar teniendo en cuenta los ejes iniciales. Es decir si tenía una disposición para mis elementos para ser vistos en registros(row) yo uso justify-content para linearlos horizontalmente, pero si cambio la dirección a columna, debo seguir usando justify-content para alinearlos verticalmente porque giró la dispoción original.
