Display Flex

En este ejercicio estoy mostrando diferentes modos de posicionamiento de elementos usando la propiedad CSS display flex

Display Flex - Default

En el display flex (solamente declarado) todos los elementos se ubican de manera lineal horizontalmente pegados unos a otros de izquierda a derecha

Display Flex - justify-content: space-around

En el display flex con la propiedad justify-content: space-around todos los elementos se ubican de manera lineal horizontalmente separados con espacios uniformes de acuerdo al ancho total del contenedor padre

Display Flex - justify-content: flex-start

En el display flex con la propiedad justify-content: space-between todos los elementos se ubican de manera lineal horizontalmente separados con espacios uniformes de acuerdo al ancho total del contenedor padre, pero teniendo en cuenta que el primer y ultimo elemento quedan pegados en los extremos del contenedor
