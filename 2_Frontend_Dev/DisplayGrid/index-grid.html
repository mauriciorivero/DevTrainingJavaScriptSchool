Display Grid

En este ejercicio estoy mostrando diferentes modos de posicionamiento de elementos usando la propiedad CSS display Grid

Display Grid - Default

En el display Grid (solamente declarado) todos los elementos se van cuadrando de manera organizando de izquierda a derecha y si llega al borde del contenedor estos bajan a la siguiente fila automaticamente. En este ejemplo tenemos una grilla de cuatro columnas y cuadro filas

Display Grid - organizado

En este ejemplo organizo 4 elementos de manera que esten en diagonal disponiendo uno a uno la posición exacta enla fila y columna correspondiente. Esto lo tengo que hacer es en cada elemento por lo que la declaración del grid-column y grid-row va en el estilo de cada elemento. Usando estas propiedades se van especificando los valore sde acuerdo a los limites horizontales o verticales segun el caso para ir posicionandolos en la grilla ejemplo para el primero grid-column: 1/2 (limtees verticales de la linea 1 y linea 2)
